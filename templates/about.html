About

Why another pastebin clone?

I know, no one needs another pastebin clone. I think dpaste, Pastie and Lodge It are amazing full featured services. I just wrote another tool like those because I was bored and thought that could be a good programming exercise for a newbie like me.

Some features

  • A clean and intuitive user interface. It means no boring forms and syntax highlight disabled by default.
  • MD5 hash for every entry preventing duplicates and auto expiry for old pastes means that database never fills up.
  • You can reply pastes, see differences and download a unified diff file.

Software used

Just few lines of Python code using web.py (anti)framework, SQLObject as database layer, SQLite to store all stuff and some HTML/CSS bullshit.
